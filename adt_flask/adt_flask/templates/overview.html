Objective

The opiod epidemic has had a serious effect on the nation's public health. Using data from the National Survey on Drug Use and Health (2015-2017), which had been converted to CSV for another project on Kaggle, this web application presents the opportunity to take a survey similar to the one used for the National Survey and browse the historical results. The goal of this project is to demonstrate practical database concepts.

Technical Details

The web application is built on Python 3.11 using Flask as a url routing mechanism and Django as an application framework. Bootstrap is used as a front-end user interface framework.

Data

Data was obtained from the following source:

  • Data from the National Survey on Drug Use and Health was obtained from this Kaggle entry: https://www.kaggle.com/datasets/thedevastator/predicting-pain-reliever-misuse-abuse. The data was formatted as a CSV with deidentified survey results coded as integers for a project involving predictive models (used here Comparing Predictive Models of Pain Reliever Misuse and Abuse).
